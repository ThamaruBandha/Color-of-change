# Color of Change – Racial Justice Awareness Website

This project is a student-built awareness website created to highlight the importance of racial justice, equality, and human dignity.

 **Live Site:**  
https://yourusername.github.io/color-of-change-racial-justice-website/

## Purpose
The goal of this site is to:
- Educate users about racial injustice
- Encourage reflection and empathy
- Motivate people to take action in their own communities

  ## What This Website Includes

### 1. Racial Justice Fact Generator
- A dynamic fact box that displays a new racial justice–related fact every time the user clicks a button
- Facts rotate through an array using JavaScript logic
- Includes subtle animations and transitions to make the interaction feel thoughtful and engaging
- Designed to promote learning through small, repeatable interactions

### 2. “Take Action” Message Generator
- An interactive tool where users enter:
  - An issue they care about
  - Their city or region
- JavaScript generates a personalized action message encouraging awareness and accountability
- Input validation ensures meaningful user interaction
- Output is animated for a smooth, user-friendly experience

### 3. Interactive Toolboxes
- Content is organized into clear sections and interactive boxes
- Hover effects and transitions improve readability and focus
- JavaScript enhances the experience without overwhelming the user

## Pages
- **Home:** Explains why racial justice matters
- **Take Action:** Helps users generate a personal action message
- **About Me:** Shares the purpose and motivation behind creating this site

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## Key JavaScript concepts applied:
- DOM manipulation (`getElementById`, event listeners)
- Arrays to store and rotate content dynamically
- Functions to control logic and reusability
- Conditional logic for input validation
- Text generation based on user input
- UI micro-interactions (button feedback, fade-in effects)
- Typewriter-style animation for improved engagement

All functionality was implemented manually to strengthen core JavaScript understanding.

This project was built as an awareness and learning initiative.
